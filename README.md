# Wediz

## Description

위디즈(wediz)는 쉽게 사용하다(with ease)라는 뜻이며, 자주 접하는 유틸리티 (날씨앱, 계산기, 시계, 그림판, 투두리스트) 를 쉽게 사용할 수 있도록 모아둔 웹앱입니다.

## Pages

- /: 메인 페이지
- /calculator: 계산기 앱
- /clock: 아날로그 시계 앱
- /painter: 그림판 앱
- /todolist: 일정관리 앱
